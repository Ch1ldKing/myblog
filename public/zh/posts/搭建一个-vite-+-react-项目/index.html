2025年10月15日 · 1 分钟 · 147 字 · Dorianyang
目录
  • 前期准备

前期准备#

首先你需要安装 node.js，mac 使用如下命令，windows linux 请自查

brew install node

然后全局安装 pnpm（我从来不用 npm，pnpm 是最好的）

npm install pnpm -g
pnpm -v

然后在项目路径中使用 vite 初始化项目

pnpm create vite . -- --template react-ts
